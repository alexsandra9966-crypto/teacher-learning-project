# Проект: Обучение с учителем

## Описание проекта
Интернет-магазин «В один клик» продаёт товары различных категорий: товары для детей, для дома, бытовую технику, косметику и продукты. 
Анализ показал снижение покупательской активности постоянных клиентов, при этом привлечение новых пользователей становится менее эффективным.

В связи с этим основной задачей является удержание текущих клиентов за счёт персонализации предложений.

## Цель исследования
Построить модель машинного обучения, которая прогнозирует вероятность снижения покупательской активности клиента в ближайшие три месяца.

## Задачи
- Подготовка целевого признака
- Предобработка и анализ данных
- Построение модели классификации
- Оценка качества моделей и выбор лучшей
- Сегментация клиентов
- Разработка рекомендаций

## Используемые методы
- Предобработка данных
- Исследовательский анализ данных (EDA)
- Модели классификации (Logistic Regression, Decision Tree и др.)
- Оценка качества (ROC-AUC, F1-score)

## Результаты
Была обучена модель, позволяющая прогнозировать снижение активности клиентов. 
Результаты могут быть использованы для персонализации маркетинговых предложений и повышения удержания клиентов.
